# MyButton

Componente de botón con diferentes especializaciones.

#### Example

<Canvas of={MyButton.Primary} />

## Usage

```ts
import {MyButton} from 'milibreria';

function Example(){
 return (
 <MyButton
 text: "Primer botón",
 txtcolor: 'white',
 bgcolor: 'red',
 bordercolor: 'black',
 borderwidth: '2px',
 hoverTxtcolor: 'black',
 hoverBgcolor: 'white',
 disabled: false,
 size: "large",
 onClick: () => alert("Primer botón"),
 />
 );
};

export default Example;
```
